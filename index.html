Projetos

AutoCelesc

Robô de automação que baixa faturas de energia de dezenas de unidades automaticamente, eliminando o trabalho manual de acessar o portal e baixar documento por documento. Desenvolvido em Python com Playwright, o sistema faz login, navega entre as unidades e lida com o carregamento dinâmico das páginas. Conta com um mecanismo de retomada que salva o progresso e continua de onde parou caso a conexão seja bloqueada, além de tratar casos de unidades sem fatura sem interromper a execução

Task-Flow Web

Aplicação web de gerenciamento de tarefas com interface própria, construída em Python com Flask. O projeto tem foco em segurança e boas práticas: senhas protegidas com hashing, prevenção contra SQL injection, proteção CSRF e controle de sessão. A arquitetura usa organização em módulos (Blueprints) para manter o código limpo e escalável

Task-Flow API

API REST de gerenciamento de tarefas onde cada usuário acessa apenas os próprios dados, com autenticação segura. Construída em Python com FastAPI, usa autenticação JWT, validação de dados e controle de acesso por usuário em todos os endpoints. Inclui uma suíte de testes automatizados com pytest e está publicada em produção.
