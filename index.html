AutoCelesc

Robô que baixa automaticamente as faturas de energia de dezenas de unidades, eliminando o trabalho manual de acessar o portal documento por documento. Faz login, navega entre as unidades e lida com o carregamento dinâmico das páginas. Conta com mecanismo de retomada que salva o progresso e continua de onde parou caso a conexão seja bloqueada, tratando unidades sem fatura sem interromper a execução.

Task-Flow Web

Aplicação web de gerenciamento de tarefas com interface própria, construída em Flask. Foco em segurança e boas práticas: senhas protegidas com hashing, prevenção contra SQL injection, proteção CSRF e controle de sessão. Arquitetura organizada em módulos (Blueprints) para manter o código limpo e escalável.

Task-Flow API

API REST de gerenciamento de tarefas onde cada usuário acessa apenas os próprios dados, com autenticação JWT, validação de dados e controle de acesso por usuário em todos os endpoints. Inclui suíte de testes automatizados com pytest e está publicada em produção.
